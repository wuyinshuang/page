## ✨ 功能特性

- ✅ **响应式设计** - 适配手机、平板、桌面设备
- ✅ **暗色主题** - 支持暗色/亮色主题切换
- ✅ **文章分页** - 首页自动分页展示
- ✅ **SEO 优化** - 自动生成站点地图和 RSS
- ✅ **代码高亮** - 支持多种编程语言
- ✅ **访问统计** - 集成不蒜子访问统计
- ✅ **评论系统** - 支持添加评论功能

## 📄 文章模板

文章使用标准 Markdown 语法，支持：

- 标题、段落、列表
- 代码块（支持语法高亮）
- 表格、引用
- 图片、链接
- 数学公式（LaTeX）

## 🎨 主题定制

博客基于 Jekyll 的 Minima 主题，可通过修改以下文件自定义：

1. **修改样式**：编辑 `assets/css/style.css`
2. **修改布局**：编辑 `_layouts/` 下的布局文件
3. **修改配置**：编辑 `_config.yml`
4. **修改组件**：编辑 `_includes/` 下的组件文件

## 🔧 技术栈

- **静态网站生成器**：[Jekyll](https://jekyllrb.com/)
- **托管服务**：[GitHub Pages](https://pages.github.com/)
- **前端框架**：纯 HTML/CSS/JavaScript
- **CSS 预处理器**：Sass/SCSS
- **图标库**：Font Awesome
- **字体**：Inter、JetBrains Mono
